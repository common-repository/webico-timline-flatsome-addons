== Installation ==
 
This section describes how to install the plugin and get it working.
From your WordPress dashboard
=1. Visit 'Plugins > Add New'
=2. Search for WBC Timeline.
=3. Activate WBC Timeline from your Plugins page.Cài đặt plugin -> Vào đăng post timeline -> Vào flatsome UX Builder chọn element WBC Timline và cấu hình tương ứng 
Sử dụng shortcode [wbc-timline cat='']
